# Desafios de Lógica de Programação - DIO

Este repositório contém os desafios realizados no curso de Lógica de Programação da DIO.

## Desafios

### 1. Classificador de Nível de Herói
- **Descrição**: Neste desafio, o código classifica o nível de um jogador com base no valor de XP.
- **Como usar**: 
- Execute o arquivo `desafio1.js`ou `desafio1.py`, forneça no código o nome do jogador e o seu respectivo valor de XP.

### 2. Calculadora de Partidas Rankeadas
- **Descrição**: Neste desafio, o código calcula o saldo de partidas ranqueadas de um herói com base no número de vitórias e derrotas, e em seguida classifica o nível do herói de acordo com a quantidade de vitórias obtidas.
- **Como usar:**
- Execute o arquivo `desafio2.py`, informe, quando solicitado, o número de vitórias e derrotas do herói.

### 3. Classificador de Herói por Tipo (POO)
- **Descrição:**: Neste desafio, o código permite que o usuário escolha o tipo de herói e informe seu nome. Com base na escolha, o programa cria um personagem utilizando e exibe suas características, como tipo, idade e tipo de ataque.
- **O desafio possui duas versões:**
    -Versão 1: Implementada utilizando estruturas condicionais (if / elif / else).
    -Versão 2: Implementada utilizando Programação Orientada a Objetos (POO) e dicionários, tornando o código mais organizado e escalável.
- **Como usar:**
    - Execute o arquivo `desafio3_v1.py` para a versão com if/else.
    - Execute o arquivo `desafio3_v2.py` para a versão otimizada com POO.
    - Escolha o tipo do herói digitando o número correspondente, informe o nome do personagem e veja as informações exibidas no terminal.
